# WriteScript
This is a handwritten font made by me. There are font types such as Regular (basic, regular type) and Plus (more like medium, extended with e.g, ß, ő, ű, more math symbols, etc). Feel free to use it in apps like Microsoft Word.

# Installation
- Windows - Manual Copy / Font Viewer
  - Installing with Font Viewer
     > **To install, download the .ttf, open it, click on install and done.**
  - Installing with manual file copy
     > **To install, download the .ttf and copy it, go to C:\Windows\Fonts (you can replace C with the actual Windows drive letter), paste it and done.**

- iOS - iFont
  > **To install, download the .ttf, open iFont (make sure you have it, if not download it in the App Store), click on import, select the choose from files, go to the directory you saved the font, select it, import, close the dialog, you would see an install button alongside the type of the font, click on it and done.**

- macOS - Font Viewer
  > **To install, download the .ttf, open it, click on install and done.**

- Linux - User-only / System-wide
  - User-only
    > **To install, open the terminal and type:**
    ```
    mkdir -p ~/.local/share/fonts
    cp (replace with the font's filename here) ~/.local/share/fonts/
    fc-cache -fv
    ```
  - System-wide
    > **To install, open the terminal and type (NOTICE: these commands require administrator passwords-permissions!):**
    ```
    sudo cp WriteScript.ttf /usr/share/fonts/truetype/
    sudo fc-cache -fv
    ```

- Android - zFont 3
  > **To install, download the .ttf, open zFont 3 (make sure you have it installed, or get it in the Play Store), go to Local, select the .ttf file, tap on apply, enter your Android phone's brand and restart (if required), and done.**

# Comparison
> ### **Basically, use Regular, if you want simple, basic usage, and use Plus, if you want better multi-language support and more math symbols.**
| Feature / Use case | Recommended font type |
|--------------------|-----------------------|
| Simplicity | Regular |
| Pro work | Plus |
| Basic, simple characters | Regular |
| Multilingual characters, more math symbols | Plus |
| Basic Latin | Regular |
| More Unicode characters | Plus |
| English-only / Personal websites | Regular |
| Multilingual websites | Plus |
| Simple texts / Simple essays | Regular |
| Advanced texts / Advanced essays (mostly if multilingual) | Plus |
### ***Have fun using this handwritten font!***
